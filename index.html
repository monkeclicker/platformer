Welcome to a Platformer Game






The Keys






  • Jump: W, Up Arrow, Space




  • Left: A, Left Arrow




  • Right: D, Right Arrow





The Goal




    The goal of this game is to beat the level(s). Although there is only one level currently, Charles is working his hardest to create as many levels to play as possible!
